# Utilisation du code :

Génération d'une trajectoire à partir de plusieurs paramètres, A, B, théta et V.

```

B = (5, 5, 50)
V = 10  # V != 0
theta = 0
traj = Trajectoire(A, B, theta, V)
traj.simulation()
traj.display()
```
